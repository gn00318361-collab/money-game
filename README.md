# 💰 理財生存戰

> 青溪國小親職教育日 — 金融教育互動遊戲

## 🎮 遊戲介紹

「理財生存戰」是一款專為國小中年級學生設計的理財模擬遊戲，讓孩子透過角色扮演體驗賺錢、消費、投資與風險管理的基本概念。

遊戲中有一位理財助教「小王」全程陪伴，根據玩家的選擇即時給予回饋與建議。

## 🕹️ 玩法說明

1. **選擇職業** — 三種不同起點與風格：
   - 🛵 **外送員**（快節奏）：時薪高但壓力大
   - 💼 **行政員**（穩健型）：起步資金多，節奏穩定
   - 💻 **工程師**（高風險）：進修費用高但報酬成長快

2. **每月行動** — 在 12 個月內做出決策：
   - 💪 工作賺錢
   - 📚 進修提升時薪
   - 📈 買入存股（長期投資）
   - 🍜 放鬆休息（降低壓力）

3. **目標** — 在不讓壓力爆表的情況下，累積資產達到 $180,000！

## ⚠️ 遊戲機制

- **壓力系統**：工作和進修都會增加壓力，壓力過高（>80%）會影響行動
- **投資回報**：存股需要 $30,000，帶來被動收入
- **助教提示**：即時回饋幫助孩子理解每個決策的影響
